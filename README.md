# parking-control
Projeto de controle de estacionamento de um condomínio (sem regras de negócio específicas) com finalidade de aprendizado de construção de API.

Foram utilizados:   
JDK11;  
Maven;  
Springboot;  
JPA e Hibernate;  
Postman;  
pgAdmin 4;  
Spring Tool Suite 4;
